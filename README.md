## Installation

To install HR_Web app, follow these steps after cloning the repository:

Install the venv if not already having it. 
```
python3 -m venv venv 
```

Activate the venv environment
```
. myenv/bin/activate
```

Install the required modules
```
pip install -r requirements.txt
```

To start the app
```
export FLASKAPP=app
flask run 
```
